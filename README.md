# Публикация изображения в Telegram(publish_photo.py) #

Этот скрипт позволяет вам публиковать фотографии из указанного каталога в канале Telegram или чате. Вы можете выбрать определенную фотографию или позволить скрипту случайным образом выбрать одну из каталога.

## Требования
* python 3.x;
* Необходимые библиотеки:
```
python-dotenv==1.0.1;
python-telegram-bot==13.0;
urwid==2.6.16;
urllib3==1.26.15;
request==2.32.3;
```
* Установите необходимые библиотеки с помощью команды pip:
```
pip install "Библиотека"
```

### Пример: ###

```pip install python-telegram-bot```

## Настройка ##

1. Создать бота Telegram

* Откройте Telegram и найдите «BotFather».
* Начните чат с BotFather и используйте ```/newbot``` для создания нового бота.
* BotFather предоставит вам **токен** бота . Сохраните этот токен.

2. Получите свой идентификатор чата

* Начните чат с вашим ботом в Telegram.
* Отправьте сообщение вашему боту.
* Для получения идентификатора чата используйте следующий URL-адрес:
```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
```

Замените ```<YOUR_BOT_TOKEN>``` на ваш настоящий токен бота.

* Найдите ```chat``` в ответе раздел, содержащий ваш **CHAT_ID** .

3. Создать .envфайл

Создайте ```.env``` файл в том же каталоге, что и скрипт, со следующим содержимым:

```
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
PHOTO_DIR=path_to_photos
```
* Замените ```your_bot_token``` токеном от BotFather.
* Замените ```your_chat_id``` на идентификатор чата, полученный выше.
* Укажите ```PHOTO_DIR``` путь к каталогу, содержащему фотографии для публикации.

## Как запустить скрипт ##

Запустите скрипт с помощью следующей команды:

```
python telegram_photo_publisher.py
```
При желании вы можете указать конкретную фотографию для публикации:

```
python telegram_photo_publisher.py --photo /path/to/photo.jpg
```
Если фотография не указана, скрипт случайным образом выберет одну из предоставленного каталога (```PHOTO_DIR```).

## Результат ##

После запуска скрипта проверьте свой канал или чат Telegram на наличие отправленной фотографии.

Скрипт выведет сообщение в терминал, если фотография была успешно отправлена.

# Загрузка изображений SpaceX(fetch_spacex_images.py) #

Этот скрипт Python позволяет вам загружать изображения с запусков SpaceX с помощью API SpaceX . Вы можете указать конкретный **идентификатор** запуска или использовать ```«latest»``` - для получения изображений с самого последнего запуска.

## Как запустить скрипт ##

Запустите скрипт с помощью следующей команды:

```
python fetch_spacex_images.py --spacex_id latest --dir images
```
Эта команда загрузит изображения с последнего запуска SpaceX и сохранит их в ```images``` каталоге.

## Аргументы ##
* ```--spacex_id``` (необязательно): Идентификатор запуска SpaceX. Вы можете указать конкретный идентификатор запуска (например, ```1``` или ```flight_number```) или использовать ```latest``` для получения изображений с последнего запуска (по умолчанию ```latest```).
* ```--dir```(необязательно): Каталог, в котором будут сохранены изображения. По умолчанию — images.

## Пример ##
```
python fetch_spacex_images.py --spacex_id latest --dir downloaded_images

```
Это позволит загрузить изображения с последнего запуска SpaceX и сохранить их в ```downloaded_images``` каталоге.


# Загрузка изображений NASA APOD (fetch_nasa_apod.py) #

Скрипт загружает изображения с сервиса "Astronomy Picture of the Day".

## Настройка ##

1. Получите ключ API NASA

Перейдите на сайт [API NASA](https://api.nasa.gov) и зарегистрируйтесь, чтобы получить свой ключ API.

2. Создать ```.env``` файл (необязательно)

Если вы предпочитаете не указывать ```.env``` в скрипте, вы можете создать файл для безопасного хранения своего ключа API.

## Как запустить скрипт ##

Запустите скрипт с помощью следующей команды:

```
python fetch_nasa_apod.py --api_key YOUR_API_KEY --count 5 --dir images
```
Замените ```YOUR_API_KEY``` его вашим реальным ключом API NASA.

## Аргументы ##
* ```--api_key```(обязательно): Ваш ключ API NASA.
* ```--count```(необязательно): Количество изображений, которые вы хотите загрузить. По умолчанию 5.
* ```--dir```(необязательно): Каталог, в котором будут сохранены изображения. По умолчанию — images.

## Пример ##
```
python fetch_nasa_apod.py --api_key YOUR_API_KEY --count 10 --dir downloaded_images
```
Это позволит загрузить 10 изображений и сохранить их в ```downloaded_images```каталоге.
Изображения будут названы ```nasa_apod_0```, ```nasa_apod_1``` и т. д. с указанием правильного расширения файла.

# Автопубликация всех изображений в Telegram (auto_publish.py) #
Этот скрипт Python автоматически публикует фотографии из указанного каталога в канале Telegram или чате через регулярные промежутки времени.

## Настройка Telegram-бота ##

1. **Создать бота Telegram**

* Откройте Telegram и найдите «BotFather».
* Начните чат с BotFather и используйте ```/newbot``` для создания нового бота.
* BotFather предоставит вам **токен** бота . Сохраните этот токен.

2. **Получите свой идентификатор чата**

* Начните чат с вашим ботом в Telegram.
* Отправьте сообщение вашему боту.
* Для получения идентификатора чата используйте следующий URL-адрес:

```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
```
Замените ```<YOUR_BOT_TOKEN>``` на ваш настоящий токен бота.

* Найдите ```chat``` в ответе раздел, содержащий ваш **CHAT_ID** .

3. Создать ```.env``` файл

Создайте ```.env``` файл в том же каталоге, что и скрипт, со следующим содержимым:

```
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
PHOTO_DIR=path_to_photos
PUBLISH_INTERVAL=14400  # Interval in seconds (default: 4 hours)
```
* Замените ```your_bot_token``` токеном от BotFather.
* Замените ```your_chat_id``` на идентификатор чата, полученный выше.
* Укажите ```PHOTO_DIR``` путь к каталогу, содержащему фотографии для публикации.
* Установите ```PUBLISH_INTERVAL``` желаемый интервал между публикацией фотографий (по умолчанию 4 часа).

## Как запустить скрипт ##

Запустите скрипт с помощью следующей команды:

```
python telegram_photo_publisher.py
```

При желании вы можете указать пользовательские значения для каталога фотографий и интервала:

```
python telegram_photo_publisher.py --photo_dir /path/to/photos --interval 3600
```
Это позволит задать каталог фотографий и интервал (в секундах) между публикацией каждой фотографии.

## Результат ##
Скрипт будет непрерывно перебирать фотографии в указанном каталоге, отправляя их в указанный чат Telegram с заданным интервалом.



# Загрузка изображений NASA EPIC (fetch_nasa_epic.py) #
Этот скрипт Python позволяет извлекать изображения Земли из API камеры (EPIC) NASA, загружать их и сохранять в указанном каталоге.

## Как использовать ##
1. **Получить ключ API NASA**

Для доступа к API NASA вам нужен ключ API. Вы можете получить его, зарегистрировавшись на сайте [API NASA](https://api.nasa.gov/).

2. **Запуск скрипта**

Для запуска скрипта используйте следующую команду:

```
python fetch_nasa_epic.py --api_key YOUR_API_KEY --count 5 --dir images
```
Замените ```YOUR_API_KEYего``` вашим реальным ключом API NASA.

## Аргументы: ##

* ```--api_key```(обязательно): Ваш ключ API NASA.
* ```--count```(необязательно): Количество изображений, которые вы хотите загрузить. По умолчанию 5.
* ```--dir```(необязательно): Каталог, в котором будут сохранены изображения. По умолчанию - ```images```.

## Пример: ##
```
python fetch_nasa_epic.py --api_key YOUR_API_KEY --count 10 --dir downloaded_images
```

Это позволит загрузить 10 изображений и сохранить их в ```downloaded_images``` каталоге.

# Загрузчик изображений(utils.py) #

## Как запустить скрипт ##

Запустите скрипт с помощью следующей команды:

```
python image_downloader.py "https://example.com/image.jpg"
```

## Аргументы ##
* ```url``` - адрес ссылки(обязательно)
* ```--output``` - выбор папки для сохранения фото

## Пример ##

```
python image_downloader.py "https://example.com/image.jpg" --output "/path/to/save/image.jpg"
```
Это загрузит изображение и сохранит его в формате ```/path/to/save/image.jpg```.


Изображение будет сохранено под своим исходным именем или под именем по умолчанию, например ```downloaded_image.jpg```.

Если изображение не загружается, убедитесь, что URL-адрес указан правильно и изображение находится в открытом доступе.
Убедитесь, что выходной каталог существует или может быть создан (скрипт автоматически создаст каталоги при необходимости).
